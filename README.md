# Distributional Deep RL Market Maker

> 10-week research project — University of Chicago  
> Building and scientifically evaluating a distributional reinforcement learning market maker in a high-fidelity limit order book simulator.

## Project Overview
This project builds a distributional RL market maker using QR-DQN and IQN with a CVaR tail-risk objective, evaluated against classical analytical benchmarks (AS, GLFT) across multiple market regimes inside an ABIDES-Gym simulator calibrated to real LOBSTER tick data.

## Research Questions

1. Why distributional RL for market making?
2. Can QR-DQN/IQN outperform analytical market-making models such as AS and GLFT?
3. How does risk sensitivity (CVaR α) alter market-making behavior?
4. Which state representation performs best (Handcrafted, CNN, AE, LSTM)?
5. How robust are distributional agents during stress events and regime shifts?
6. Does temporal memory (LSTM) provide measurable benefit in realistic LOB environments?
7. Can distributional RL improve tail-risk management without sacrificing excessive return?
